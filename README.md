# Otenki Buzzer

Informs you today's weather when you reached near device.

## Demo

See [https://www.instagram.com/p/BGrRCrkzQzU/](https://www.instagram.com/p/BGrRCrkzQzU/).

## Requirements

Install font:

```sh
$ sudo apt-get install ttf-kochi-gothic
```

Connect buzzer to GPIO 21.
